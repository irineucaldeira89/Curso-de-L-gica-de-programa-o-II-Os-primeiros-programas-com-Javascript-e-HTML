Para ser aprovada em um teste de emprego, uma pessoa foi submetida ao seguinte teste de lógica: a = 100 b = 'palavra' c = -50 if (b.length > 7) { console.log(b) } else { console.log(a + c) } Com base no trecho de código acima, marque a alternativa correta a respeito do resultado esperado. *** Resposta Correta: C: A saída do console será 50. Como a palavra contida na variável b possui 7 letras e 7 não é maior que 7, é igual, a condição else será executada. Nela, executamos a seguinte operação: 100 -50 que é 50.

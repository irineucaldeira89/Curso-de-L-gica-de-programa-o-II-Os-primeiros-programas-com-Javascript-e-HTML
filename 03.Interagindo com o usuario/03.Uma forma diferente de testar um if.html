Usamos inclusive o operador ==. No entanto, podemos simplificar esse código para: Veja que não usei mais o operador ==. Você deve estar intrigado por ter funcionado. Se aprendi já é true, é redundante realizarmos o teste aprendi == true. Se uma variável já guarda true ou false podemos usá-la diretamente no if poupando alguns caracteres.

Uma função pode conter muitos parâmetros ou nenhum, como ilustram os códigos: function atualizaTela() { ... } function desenhaCirculo(x, y, raio) { ... } Sabendo disso, analise as afirmações abaixo e marque as verdadeiras em relação às funções no Javascript. *** Resposta Correta: Sempre colocamos os parâmetros entre os parênteses, quando necessário. Nem toda função tem parâmetro. Porém, caso as funções precisem de um parâmetro, eles sempre serão adicionados entre os parênteses.

Quando não definimos a cor do stroke a cor padrão utilizada é black. No entanto, para mudarmos a cor do stroke para uma cor diferente de preto, precisamos realizar essa operação através da propriedade strokeStyle Exemplo: pincel.strokeStyle = 'red' O código acima adotará a cor vermelha para o stroke.
